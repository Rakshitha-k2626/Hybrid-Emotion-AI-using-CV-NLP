# 🧠 Hybrid Emotion AI System

A multimodal AI system that combines Computer Vision and Natural Language Processing to detect human emotions.

## 🚀 Features
- Emotion detection from facial expressions (FER)
- Emotion classification from text (Naive Bayes NLP model)
- Fusion of both signals for final decision
- Handles mixed emotional states

## 🧠 Tech Stack
- Python
- OpenCV
- FER (Deep Learning)
- Scikit-learn (NLP)
- Jupyter Notebook

## 📌 What I learned
- Multimodal AI systems
- Emotion recognition techniques
- Model fusion logic
- Real-world AI limitations
